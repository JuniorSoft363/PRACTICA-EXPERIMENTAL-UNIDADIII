\section{Introduction}

El desarrollo de aplicaciones web modernas exige comprender tanto los mecanismos internos que gestionan el estado de una petición HTTP como las decisiones arquitectónicas que determinan la escalabilidad, el mantenimiento y el rendimiento de un sistema. La presente investigación aborda, con rigor científico-bibliográfico y respaldo de fuentes académicas de acceso abierto (IEEE Xplore, ACM Digital Library, arXiv, MDPI, DOAJ y revistas indexadas), los cuatro temas que conforman la Unidad 3: los objetos integrados para la gestión de estado, el acceso a datos desde aplicaciones web, los patrones de arquitectura de software y el diseño de arquitecturas web escalables. El documento sigue el formato de citación IEEE, referenciando cada fuente utilizada al final del trabajo.
